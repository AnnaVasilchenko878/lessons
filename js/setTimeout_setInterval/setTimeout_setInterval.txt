
- задержка измеряется в миллисекундах
- аргументы необязательно передавать
- таймер не учитывает время работы функции внутри него, для этого использую рекурсивный setTimeout
- независимые анимации лучше группировать
- не слкдует перерисовывать анимацию если закрыта вкладка браузера

setTimeout(call-back, задержка, аргументы)

  Записать числовой инентификатор таймера
  перемен = setTimeout(call-back, задержка)

  Убрать setTimeout
    clearTimeout(id таймера)

setInterval - это функция которая выполняет другую функцию с зпдпнным интервалом

setInterval(call-back, интервал в миллисекундах)

Сгруппировать анимации 
setInterval(function(){
  animation1()
  animation2()
})

requestAnimationFrame() - метод браузера который используется с целью планирование анимации с учетом производительности (оптимизация ресурсов, скрытие вкладки, без мерцания)

Запуск 
let requestId = requestAnimationFrame(call-back)


Пармаетры 
time - расчет времени обновления экрана
