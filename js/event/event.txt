Терминология 
 1. window - глобальный объект браузера, окно и контекст выполнения js
 2. document - это свойство объекта window который предоставляет браузер, он включает в себя полность весь html, css, js
 3. Делегирование - назначение функуции родительскому блоку которая потом переходит потомкам (всплытие, перехват)
 4. Всплытие событий - механизм который позволяет всплывать событиям от вложенных к родительским.


Ньюансы 
- привязанный обработчик событий к document срабатывает на всех потомках любой вложенности
- делегирование позволяет реализовать всплытие и перехват событий
- всплытие можно остановить
- если имеется множества кнопок с разными действиями при клике, лучше обработать их через класс

Добавить обработчик события
1. имя_эл-та.имя_обработчика = function(){} (Перезаписывается)
2. имя_эл-та.addEventListener('имя_события', 'имя_функции')

Добавить обработчик в html (иметь в виду но не использовать)
<тег имя_обработчика="тело кода"></тег>

Отправить форму
эл-т.addEventListener()

Найти тег на который произошло событие 
event.target.tagName

Найти близжайщего родителя цели, на которой стоит обработчик, если цели нет то null
event.target.closest('цель')

Получить атрибут цели (undefined если нет)
event.target.dataset.action

Получить значение атрибута 
event.target.value
