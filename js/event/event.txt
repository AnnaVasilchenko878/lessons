Терминология 
 window - глобальный объект браузера, окно и контекст выполнения js
 document - это свойство объекта window который предоставляет браузер, он включает в себя полность весь html, css, js
 Делегирование - назначение функуции родительскому блоку которая потом переходит потомкам

Ньюансы 
- привязанный обработчик событий к document срабатывает на всех потомках любой вложенности

Добавить обработчик события
1. имя_эл-та.имя_обработчика = function(){} (Перезаписывается)
2. имя_эл-та.addEventListener('имя_события', 'имя_функции')

Добавить обработчик в html (иметь в виду но не использовать)
<тег имя_обработчика="тело кода"></тег>

Отправить форму
эл-т.addEventListener()
